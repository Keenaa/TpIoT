This is the repository for Highsoft's JavaScript charting libraries: Highcharts, Highstock and Highmaps.

The libraries are available under different licenses depending on whether it is intended for commercial/government use, or for a personal or non-profit project.

Read more on licensing alternatives here:
- Highcharts: https://shop.highsoft.com/highcharts
- Highstock: https://shop.highsoft.com/highstock
- Highmaps: https://shop.highsoft.com/highmaps
